Reflection

What I learned from this class:

  • How to set up a server for creating websites
  • How to create the beginnings of a website with the basics of HTML (!doctype html, html, head, body)
  • How to use tags and attributes within HTML like headings, paragraphs, page breaks, ul, ol, images, etc.
  • How to create tables using table, th, td, & tr
  • How create an image map using map & area
  • The basics of CSS such as adding color, different fonts, positions, changing width & height, padding, margins, etc.
  • The basics of Bootstrap and how to use and adjust it with CSS
  • How to create an image simply using div, and how to the HTML instead of using style
  • How to create a navigation menu only using ul and li
  • The basics of responsive design for smaller screens such as phones and tablets.
  • How psuedo-classes, child & sibling selectors, psuedo elements, and attribute selectors are used in CSS
  • How to use and create animation in CSS and implement it into a webpage
  • Different display options such as display: flex, display: block, position: absolute, float: left, etc.
  • The basics of Javascript: arrays, loops, arrays, operators, random numbers, DOM, etc.
  • How to create simple games using Javascript

How this class has helped me:

I've learned how to combine basic computer programming with creativity allowing me to create websites and webpages exactly how I choose to make and share them. I'm able to create simple and more complex designs with responsive or non-responsive design. I have really enjoyed taking this class because it has strengthened my programming skills; I've previously learned very basic HTML, but it was great taking a class and getting a more solid grounding on it along with learning CSS and Javascript.
